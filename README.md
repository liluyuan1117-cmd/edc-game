- ✅ 粒子特效 - 按压时爆开彩带

## 🚀 快速开始

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/liluyuan1117-cmd/edc-game.git
cd edc-game

# 直接用浏览器打开
open index.html
```

### 部署到 GitHub Pages
1. 仓库 Settings → Pages
2. Source 选择 `Deploy from a branch`
3. Branch 选择 `main`，文件夹 `/ (root)`
4. Save 后访问 `https://你的用户名.github.io/edc-game/`
